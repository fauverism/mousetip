What is Mousetip?

Mousetip is a tooltip layer for keyboard shortcuts built on top of the Mousetrap library. It provides a simple way for binding keyboard shortcuts and dynamically displaying tooltips for applications that heavily rely on them.

Why use it?

If your application has a separate page for explaining keyboard shortcuts, then Mousetip is the perfect fit! Even if you have just a few keyboard shortcuts in your application, it's easy to provide the user with non-intrusive, dynamic tooltips for them.

How do I add Mousetip to my application?

Simple!

<script src="/path/to/mousetrap.js"></script>
<script src="/path/to/mousetip.js"></script>
<script src="/path/to/mousetip.ui.js"></script>
